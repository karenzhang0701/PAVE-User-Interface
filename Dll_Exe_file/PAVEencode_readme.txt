Usage 1:
    PAVEencode.exe curve_definition_file.txt
	
This will take "curve_definition.txt" file as input, encode the file and output as "curve_definition.bin" file.
	

Usage 2:
    PAVEencode.exe curve_definition_file.txt -decode curve_definition_file.bin
	
This will take "curve_definition_file.bin" as input, decode the file and output as "curve_definition_file.txt". Note the file names of .txt and .bin may be different.
