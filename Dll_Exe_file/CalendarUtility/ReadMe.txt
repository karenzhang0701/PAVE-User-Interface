1. Generate calendar holiday .txt files

HolGenerator.exe: This utility is used to generate the calendar holiday txt file from the csv files. The csv files are generated by Findur.

Usage:
	HolGenerator.exe  "Calendar" "StartDate" "EndDate" "CSV file"
	
It will generate the txt file "Static{Calendar}HolDates.txt"


e.g.
	HolGenerator.exe  ADB 20000101 21200101 ADBHolidaySchedule_FINDUR.csv
	
It generates file "StaticADBHolDates.txt". For all calendar please see "Generate_txt.bat" file.



2. Encode the .txt files to .bin files

PAVEencode.exe: This utility encodes the .txt files to .bin files

Usage:
	PAVEencode.exe ".txt file"
	
e.g.
	PAVEencode.exe StaticADBHolDates.txt
	
It will convert "StaticADBHolDates.txt" file to "StaticADBHolDates.bin" file.
